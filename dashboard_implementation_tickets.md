# ダッシュボード実装チケット（50%→80%→100%）

## 運用ルール
- チケットID: `MVP-xx` / `EXP-xx` / `FIN-xx`
- 優先度: `High` / `Medium` / `Low`
- ステータス: `TODO` / `DOING` / `DONE`
- 完了条件: 各チケットの「受け入れ条件」を満たしたら `DONE`
